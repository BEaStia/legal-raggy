# Definition of Done

- [ ] Критерии task выполнены небольшим связанным изменением.
- [ ] Tests, lint, types, security, container smoke-test и docs audit прошли либо помечены неприменимыми с причиной.
- [ ] Review относится к текущему SHA; открытых P0/P1 нет, P2 исправлены или приняты явно.
- [ ] Матрица трассировки, ADR, radar, risks и handoff синхронизированы.
- [ ] Статья написана по [`.articles/STYLEGUIDE.md`](../.articles/STYLEGUIDE.md) и ссылается на доказательства, не дублируя их.
- [ ] Рабочее дерево не содержит случайных файлов.
