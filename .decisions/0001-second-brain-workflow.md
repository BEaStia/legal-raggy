# ADR-0001: Keep a repository-local second brain

- Date: 2026-06-28
- Status: accepted

## Context

Проект начинается с подробного brief и будет выполняться последовательными агентными сессиями. Контекст чата недолговечен, а отдельный облачный трекер создаёт второй источник истины.

## Options

Использовать только чат; завести внешний трекер; хранить Markdown рядом с кодом.

## Decision

Храним задачи, решения, radar, handoff, reviews, risks, DoD, трассировку и статьи в Git. Статус task определяется каталогом, WIP равен одному, документация проверяется hooks и CI.

## Consequences

Контекст переносим и обозрим; изменения reviewable. Цена — дисциплина обновления и небольшой локальный аудитор.
