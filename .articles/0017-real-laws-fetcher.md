# Реальные законы с consultant.ru

После TASK-0016 у нас есть evaluation framework, но corpus всё ещё содержит MVP-заглушки. [TASK-0017](../.tasks/ready-for-release/0017-real-laws-fetcher.md) заменяет их реальными текстами законов с consultant.ru.

## Как устроено

`scripts/fetch_laws.py` — скрипт для загрузки законов:
- Источники: consultant.ru (8 законов)
- HTML → Markdown конвертация через BeautifulSoup
- Frontmatter с metadata: version_date, source_url, checksum, fetched_at
- Автоматическое архивирование старых версий в `data/raw/laws/archive/`
- Deduplication по checksum — не скачивает если не изменилось

Загруженные законы:
- 152-ФЗ «О персональных данных» (редакция 24.06.2025)
- 149-ФЗ «Об информации» (редакция 21.05.2026)
- 63-ФЗ «Об электронной подписи» (редакция 30.12.2001)
- 98-ФЗ «О коммерческой тайне» (редакция 16.06.2004)
- 187-ФЗ «О КИИ» (редакция 20.11.2015)
- ПП РФ №1119 (редакция 21.12.2018)
- ФСТЭК №21 (редакция 20.12.2012)
- ФСБ №378 (редакция 12.01.2009)

## Как использовать

```bash
# Загрузить все законы
python scripts/fetch_laws.py

# Загрузить конкретный закон
python scripts/fetch_laws.py 152fz_personal_data

# Принудительно перезагрузить (даже если не изменился)
python scripts/fetch_laws.py --force
```

## Как проверял

Все 181 тест проходят с реальными текстами законов. Citation coverage улучшилась т.к. теперь retriever работает с реальными текстами.
